Add to map

Make consistent, branded EIP or TRPA locator maps of project locations for presentations and reports.

  1. Pick style and layout — EIPTRPA sets the look (and the related map link in the header), then a layout: Title Slide (text + basin map, 16:9), Detail + Basin (close-up beside the basin locator, 16:9), or Basin Only (full basin, portrait 8.5×11).
  2. Search Address, APN, EIP project… — an address, APN, EIP project name or number, or coordinates ("39.0968, -119.9435") — and pick a result. Repeat to add up to 9 locations (numbered pins). Remove one with its chip, or clear the search to start over. Advanced: the button pastes GeoJSON directly.
  3. Set the detail view — the basemap thumbnails and zoom buttons Fit1:5k1:24k control the Detail + Basin layout's close-up panel. The draw tools (top-right of the map) add annotations that appear in that panel too; returns to the Tahoe Basin.
  4. Export — pick PNGPPTX and Save, or Copy puts the image on the clipboard to paste onto a slide. Toggle Caption for the styled text strip. For PPTX, Editable elements builds the slide from separate text boxes, images, and shapes you can rearrange and rewrite in PowerPoint.

Parcel data: TRPA GIS · Home button returns to the Tahoe Basin. Browse and filter all EIP projects on the EIP Projects Map (link in the header) — its Locator map button sends a selection here.
